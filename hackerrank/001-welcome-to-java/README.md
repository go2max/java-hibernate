# Printing to stdout.

```
System.out.println("Hello, World.");
System.out.println("Hello, Java.");
```
## Output Format

You must print two lines of output:

Print `Hello, World.` on the first line.
Print `Hello, Java.` on the second line.

## Sample Output

```
Hello, World.
Hello, Java.
```
